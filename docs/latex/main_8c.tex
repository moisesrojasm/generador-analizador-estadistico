\doxysubsection{Detailed Description}
Punto de entrada principal. Coordina la lectura por CLI e interfaz interactiva. 

Flujo general:\+
\begin{DoxyEnumerate}
\item El usuario elige un caso (1-\/20) o ingresa datos manualmente.
\item Se elige la distribucion y el algoritmo de ordenamiento.
\item Se mide el tiempo de ordenamiento y el de calculo de medidas por separado.
\item Se guarda una fila en resultados.\+txt con todos los datos del caso.
\end{DoxyEnumerate}
