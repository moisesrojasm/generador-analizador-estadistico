\doxysubsection{Detailed Description}
Declaraciones de las funciones para el cálculo de medidas estadísticas. 

Este módulo define las 15 medidas estadísticas requeridas en la Práctica 2, cumpliendo la regla de una función separada por cada medida. Las funciones que operan sobre datos ordenados requieren que el arreglo haya sido previamente ordenado de forma ascendente.
